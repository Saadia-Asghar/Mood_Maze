# Sound Files Directory

Place your MP3 sound files here.

## Required Files:
- `click.mp3` - Button clicks
- `success.mp3` - Success actions
- `reject.mp3` - Rejection actions
- `swipe.mp3` - Card swipes
- `complete.mp3` - Completion sounds
- `flip.mp3` - Card flips
- `reel.mp3` - Movie reel sounds

## Where to Get Sounds:
See `SOUND_FILES.md` in the project root for download links and instructions.
